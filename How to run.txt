[RU]
1. Скачайте Python по следующей ссылке [https://www.python.org/]
2. Добавьте Python в PATH (Будет галочка при установке)
3. Запустите этот код в коммандной строке: [pip install -r requirements.txt] (Находясь в директории с файлами проекта)
Или запустите этот код в коммандной строке: [pip install opencv-python numpy mss pynput pywin32 pyautogui pyqt5] (В любой директории)
4. Запустите [main.py]
